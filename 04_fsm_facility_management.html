Basic Facility Management Machine using Logic Gates (CircuitVerse Simulation)

Fall 2023 - Junior Student - University

This is the term-project for Computer Organization course in my university. I created a simple computer using logic gates and Finite-state Moore machine on CircuitVerse, which can store the student’s information for verifying to access the university’s facilities within limited slots and duration. This is a project-based learning to apply the knowledge of finite state machine and logic gates from the course into real-life problem.
